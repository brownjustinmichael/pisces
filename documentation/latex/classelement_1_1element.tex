\subsection{Detailed Description}
This is the basic class of the code. 



 A true run will contain multiple elements tied together at the boundaries. This element contains most of the infrastructure to run the evolution of a system with an arbitrary number of terms. It does not contain the implementation to store data, however.

Calculate the arrays needed to update the system. 



 First, evaluate all the explicit plans. The class assumes each step begins in grid space. After evaluating the explicit terms in grid space, it transforms to normal space, and evaluates the explicit terms there. If the timestep has changed, the code recalculates the implicit matrix.
